## Controle de Finanças

An Expo boilerplate including TypeScript, ESlint, Prettier, Husky, Lint-staged and Jest.

## 💻 Environment

    - Node: 16.x
    - Yarn: 1.22.x
    - Expo: 6.0.x

## 💻 Dev Libraries

    - ESLint
    - Prettier
    - Jest
    - Husky
    - Lint-staged

## 📄 Instalation

    - Clone repository
    $ git clone https://github.com/AstrOOnauta/boilerplate-expo-mobile.git

    - Install dependencies
    $ yarn

    - Run application
    $ npx expo start
